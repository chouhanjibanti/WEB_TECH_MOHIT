DataTypes 
Types of DT 

DataType:- Datatype defines which type of data we are storing .

Two Types :- 
1. Primitive DT :- predefined datatype.

Number :- numeric value.
String :- collectins of characters.
Boolean :- true[1] and false[0]
null :- we can defined according to the requirement.
undefined :- only variable declared , but not assign.
bigint :- for the big value.

2. Non Primitive DT :- user defined datatype.
Array :- Array is a huge memory block used to store the diff types of elements.

Syntax :- 
let/var/const VariableName = [1,2,3,"hy",true,"by"]

Object :- Object will store the data in the key and value pair. It represent the real time object.
Syntax :- 
var/let/const ObjectName = {
    key1:value1,
    key2:value2,
    .......
    ........
}
