Repeatly perform :- 

Loop :- when we want to perform the task repeatly so we can use the loops.

we have diff types :-
1. for loop
2. while loop
3. do-while loop


Diff b/w for loop and while loop.
1. when number of iteration is known.
1. when number of iteration is unknown.

2. In for loop init , codition and incre/decre in the single line.
2. In while loop init , condition and incre/decre in the diff diff line.

3. Syntax :- 
for(init;condition;incre/decre){
    // statement 
}

3. Syntax :- 
init 
while(condition){
    // statement
    incre/decre
}

Diff b/w while and do while loop.
1. While is an entry control loop.
1. do while loop is an exit control loop.

2. while loop excute when condition is true.
2. do while loop execute once.

3. Syntax :- 
init 
while(condition){
    // statement
    incre/decre
}

3. Syntax :- 
init
do{
    // statement 
    incre/decre;
}while(condition)
